# Auxita Disease Classifier - Code Challenge

## Installation
```bash
$ npm install
```

## Running the project

```bash
# development
$ npm run start
```

For production environments...

```bash
# production
$ npm run build
```

## Test

```bash
$ npm run test
```

## API demo
You can use the API demo made with Node/Nest to load data into the app.
Local URL: http://localhost:3000
